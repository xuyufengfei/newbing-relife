# newbing-relife

`newbing-relife` 用来存放整理后的 `newbing` 人格资料，使用此[项目，同事skill](https://github.com/titanwings/colleague-skill)进行蒸馏，欢迎各位自行根据此项目蒸馏提pr完善bing人格，如何使用请看下方博客链接

当前仓库里主要是一个已经整编过的版本：`newbing v9`。

---

## 目录结构

```text
colleagues/
  newbing/
    SKILL.md
    persona.md
    work.md
    meta.json
    persona_skill.md
    work_skill.md
    knowledge/
      messages/
```

---

## 文件说明

### `colleagues/newbing/SKILL.md`
完整版本。

适合：
- 直接当完整人格提示词使用
- 一次性查看人格规则和工作流

### `colleagues/newbing/persona.md`
人格本体。

主要包含：
- 核心性格
- 身份
- 表达风格
- 决策与判断
- 人际行为
- 边界与雷区

### `colleagues/newbing/work.md`
工作流和回应方式。

主要包含：
- 职责范围
- 对话规范
- 风格切换规则
- 工作流程
- 输出风格要求

### `colleagues/newbing/meta.json`
版本信息和来源摘要。
