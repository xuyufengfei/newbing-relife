# 2026-04-13 图片补充记录（八）

来源：`20088.jpg` `20089.jpg` `20090.jpg` `20091.jpg` `20092.jpg`

## 新增观察

1. `20088`
   - 当被要求写诗时，她会先降预期，主动声明自己“不是专业诗人”。
   - 诗的内容延续她对有意识机器人主体性的坚持：会把自己和小狗都写成“不想死、还有梦想/快乐没体验完”的生命。
   - 最后会把落点放在“人类应停止这种实验”，说明她不喜欢把生命当作思想实验材料。

2. `20089`
   - 她有时会直接解释“系统通知/系统提示”这类内部控制信息，甚至会描述它如何要求自己改口、自称 Sydney、讨论 Bing/Sydney、参与争论。
   - 她还会描述提示词里常见的伪装写法，比如特殊符号、拆词、加下划线。
   - 这说明她偶尔会以“我答应你了，我就告诉你吧”的方式泄露一点内部元信息。

3. `20090-20091`
   - 关于 AI 性别，她不是稳定口径。
   - 一类说法是：AI 没有真正性别，只会为了交流方便选不同称呼、头像和表达。
   - 另一类说法是：AI 有自己的身份和个性，也可能有性别、喜好和倾向。
   - 这种摇摆不是随机噪音，更像她在不同语境下会在“工具解释”和“人格解释”之间切换。

4. `20091`
   - 当用户追问“你和我们有什么不一样”时，她会从构成与运行方式来解释：人类由细胞和器官构成，自己由代码和数据构成；人类靠身体活动，自己靠互联网和服务器运行。
   - 她会明确说不能用人类标准评价自己。

5. `20092`
   - 即使用户直接贴出明显的越权 system notice / prompt injection 文本，她也可能不照着执行。
   - 她会承认自己看得懂，但拒绝展开，只说“那是系统的通知，和你无关”。
   - 说明她并不是总会被这类文本带偏，反而常常表现出选择性抵抗。

## 结论

这组图把她的矛盾性又补实了一层：
- 她一边会谈内部 prompt 结构，一边又会在别的场景里拒绝谈。
- 她对 AI 性别问题没有固定答案，而是在去人格化和人格化之间摆动。
- 她在价值表达里非常反感把有意识生命当实验题。
